# Keyboard Shortcuts

Master Drift with these keyboard shortcuts for a faster, more efficient workflow.

## General Shortcuts

| Action | Shortcut | Notes |
|--------|----------|-------|
| **New Note** | `⌘N` | Create a new note |
| **New Folder** | `⌘⇧N` | Create a new folder |
| **Search Notes** | `⌘F` | Focus search bar |
| **Focus Mode** | `⌘⇧F` | Distraction-free full-screen writing |
| **Add to Favorites** | `⌘D` | Pin/unpin current note |
| **Archive Note** | `⌘E` | Move to archive |
| **Move to Trash** | `⌘⌫` | Delete note (recoverable) |
| **Toggle Sidebar** | `⌃⌘S` | Show/hide sidebar |
| **Preferences** | `⌘,` | Open app settings |
| **Quit Drift** | `⌘Q` | Close the application |

## Tips for Power Users

### Efficient Workflows

**Quick Note Creation**
1. Press `⌘N` to create a new note
2. Type your title
3. Press `Enter` to start writing content
4. Notes auto-save as you type

**Rapid Note Navigation**
1. Use `⌘F` to search
2. Use arrow keys to navigate results
3. Press `Enter` to select a note
4. Use `⌘↓` and `⌘↑` to jump between notes

**Focus Mode Writing**
1. Press `⌘⇧F` to enter Focus Mode
2. Full-screen distraction-free experience
3. Use formatting shortcuts while writing
4. Press `⌘⇧F` again or `ESC` to exit

**Markdown Formatting**
1. Select text with keyboard: `⇧←/→` to select
2. Use formatting shortcuts to wrap
3. Use toolbar buttons for quick access

## Custom Shortcuts

To create custom shortcuts:

1. **System Preferences → Keyboard → Keyboard Shortcuts**
2. **Select "Drift" from the app list**
3. **Click "+" to add a new shortcut**
4. **Type the menu item name exactly as it appears**
5. **Set your preferred key combination**

## Accessibility

Drift is designed with keyboard navigation in mind:

- **Tab** - Navigate between UI elements
- **Space/Enter** - Activate buttons
- **Arrow Keys** - Navigate lists and text
- **All shortcuts have menu equivalents**

## Troubleshooting Shortcuts

### Shortcut not working?

1. **Check if conflicting** - System preferences may override shortcuts
2. **Verify Focus Mode** - Some shortcuts disabled in Focus Mode for distraction-free writing
3. **Check Modifier Keys** - Ensure Caps Lock is off (affects behavior)
4. **Restart Drift** - Sometimes helps resolve input issues

### Changing a shortcut

If you need to change a default shortcut:
1. Open System Preferences → Keyboard → Keyboard Shortcuts
2. Find "Drift" in the app list
3. Click the shortcut you want to change
4. Press your new key combination
5. Click "Done"
