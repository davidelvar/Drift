# User Guide

Welcome to Drift! This guide will help you get the most out of your note-taking experience.

## Getting Started

### Your First Note

1. **Create a new note** - Click the "+" button or press `‚åòN`
2. **Add a title** - Type in the title field at the top
3. **Write your content** - Start typing in the editor
4. **Auto-saves** - Your note is automatically saved as you type

### Understanding the Interface

**Three-Pane Layout:**
- **Sidebar (Left)** - Collections, folders, and tags
- **Note List (Middle)** - Your notes with search
- **Editor (Right)** - Content editing and preview

## Organizing Your Notes

### Using Folders

Folders help you organize notes by topic or project.

**Create a Folder:**
1. Right-click in the sidebar
2. Select "New Folder"
3. Give it a name and icon
4. Start adding notes to it

**Move Notes to Folders:**
1. Select a note
2. Right-click and choose "Move to Folder"
3. Select the destination folder

### Using Tags

Tags provide flexible categorization without strict hierarchies.

**Create a Tag:**
1. Select a note
2. Click the "+" in the inspector panel
3. Type a tag name
4. Choose a color
5. Press Enter

**Add Tags to Notes:**
- Click a note
- Use the inspector panel on the right
- Type tag names or select existing tags
- Click the "+" to apply

**Filter by Tag:**
- Click a tag in the sidebar
- See all notes with that tag

## Collections (Smart Folders)

Quick access to organized note groups:

- **All Notes** - Every note (except trash)
- **Favorites** - Starred/pinned notes
- **Archive** - Completed or old notes
- **Trash** - Recently deleted notes

**Favorite a Note:**
- Select a note
- Press `‚åòD` or click the star icon
- Appears in Favorites collection

**Archive a Note:**
- Right-click a note
- Select "Archive"
- Recoverable from Archive collection

## Editing Notes

### Writing Modes

**Edit Mode** - Plain markdown editing
- Focus on the content
- See formatting as markdown
- Press `‚åò1`

**Preview Mode** - Rendered markdown
- See how your note looks
- Perfect for reading
- Press `‚åò2`

**Split Mode** - Edit and preview together
- Write on the left, preview on the right
- Best for learning markdown
- Press `‚åò3`

## Searching Notes

### Basic Search

1. Click the search bar or press `‚åòF`
2. Type what you're looking for

4. Click a note to open it

### Search Tips

- **Title search** - Find notes by title
- **Content search** - Search note content too
- **Tag filtering** - Click a tag to filter
- **Folder filtering** - Click a folder to limit search scope

## Managing Notes

### Delete a Note

**Soft Delete (to Trash):**
1. Select a note
2. Press `‚åò‚å´` or right-click ‚Üí "Move to Trash"
3. Note moves to Trash collection
4. Recoverable for 30 days

**Permanent Delete:**
1. Open Trash collection
2. Right-click the note
3. Select "Delete Permanently"
4. Gone forever - be careful!

### Rename/Move Notes

**Change Title:**
- Click the title
- Edit text
- Press Enter

**Move Between Folders:**
1. Right-click the note
2. Select "Move to Folder"
3. Choose destination
4. Note updates instantly

## Tips and Tricks

### Productivity Tips

1. **Use Favorites for active work** - Pin notes you're currently working on
2. **Archive completed notes** - Keep your All Notes view focused
3. **Tag consistently** - Develop a tagging system that works for you
4. **Search before creating** - Avoid duplicates

### Writing Tips

1. **Use headings to structure** - Great for long notes and previews
2. **Leverage code blocks** - Syntax highlighting for any language
3. **Link related notes** - Use markdown links to connect ideas
4. **Block quotes for emphasis** - Great for key points

### Organization Tips

1. **Create a "Inbox" folder** - Temporary holding area for new notes
2. **Use tags for workflows** - Tags like #inprogress, #review, #done
3. **Archive yearly** - Move old notes to archive for performance
4. **Regular cleanup** - Empty trash periodically
