## For Contributors

Start here if you want to contribute:

1. **Read [DEVELOPMENT.md](DEVELOPMENT.md)** - Set up your environment
2. **Review [ARCHITECTURE.md](ARCHITECTURE.md)** - Understand the codebase
3. **Check [../README.md](../README.md)** - Contributing guidelines
4. **Create a feature branch** - `git checkout -b feature/my-feature`
5. **Submit a pull request** - We review all contributions
